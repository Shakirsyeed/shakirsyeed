### 🔴 1. Customer Churn Analysis — Telecom (2024)

**Problem:** A telecom company had 5,000+ customers with no clarity on who was about to leave — or what was driving churn across contract types and regions.

**Solution:** Built a complete end-to-end analytics pipeline — cleaned raw data, designed normalized SQL database, performed Python EDA, and delivered 6 actionable business recommendations.

| 🔧 Tools | Python (Pandas, Seaborn) · SQL · Power BI · Excel |
|----------|--------------------------------------------------|
| 📦 Dataset | 5,000 customers · 20 features · 5 SQL tables |
| 🔑 Key Findings | Month-to-Month churn 76% · $152K monthly revenue at risk |
| 💥 Impact | 2,214 high-risk customers identified and flagged |

[![View Full Project](https://img.shields.io/badge/🔗_View_Full_Project-Customer_Churn-FF5B5B?style=for-the-badge)](https://github.com/Shakirsyeed/Customer-Churn-Prediction-Telecom)

**What I did step by step:**

- 🧹 **Data Cleaning** — handled nulls, fixed data types, built 5 normalized SQL tables
- 📊 **EDA** — churn patterns by contract, region, satisfaction score, tenure cohorts
- 🗄️ **SQL Analysis** — 12 queries using JOINs, GROUP BY, views, aggregations, CTEs
- 📈 **Visualizations** — 5-sheet Excel dashboard + 5-page interactive HTML dashboard
- 💡 **Recommendations** — 6 business decisions backed by data evidence
